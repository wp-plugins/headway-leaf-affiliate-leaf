Allows users to easily add an powered by box to their site. A Headway leaf to help you promote the Headway Theme.

== Description ==

This plugin adds a new leaf for use with your Headway Theme. The leaf allows you to select a color scheme that goes with your site to help you promote the use of Headway Themes.

This plugin requires [Headway Themes](http://headwaythemes.com) to be installed and activated for it to work.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `headway-leaf-affiliate-leaf` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Remember that you need to have Headway Theme installed and activated to  use this plugin.
